### :fire: Running

```bash
# Clone the repo
$ git clone https://github.com/diegolemospadilha/mockserver.git 

# Enter in the repo
$ cd mockserver

# Install json-server globally in your machine using the 0.17.4 version
$ npm install -g npm install -g json-server@0.17.4

# Start the mock
$ npm i && npm start
```
